CSS LESSON

CSS stands for Cascading Style Sheet. While HTML builds the structure of web pages, CSS add styles to the structure/skeleton of the web pages. In other words, CSS is used to adds beauty and design to a web page. Think of the beautiful background color and other lovely designs in this application; that is the effect of CSS.

CSS can written in of the following three ways with each one having its advantages and disadavantages.

  1. Inline styling
  2. Embedded styling
  3. External styling

This is used to style HTML elements directly. This is done by adding style attribute to the opening tag of an HTML element and applying the desired styles. This is not the best way to style especially when you want to apply same style to multiple web pages, but in some cases it is necessary to use.

Writing your style rules within the head tag of the HTML document structure is called embedded styling. This is often used to style a single web page and especially where the styles rules are few. All you have to add within the head after the title tag, is add the style tag and write your css rules.

For seperation of concern, it is highly recommeded to write css in an external style sheet. To do this, create a CSS file in with the .css name extension, example style.css. And in the css file, you write all style rules. The advantage of this is that you can write single style from a css file which would apply to multiple web pages.
